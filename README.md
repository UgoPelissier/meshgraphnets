# Learning Mesh-Based Simulation with Graph Networks

## Setup @Safran™

```bash
module load conda
module load artifactory
mamba env create -f utils/envs/graphnet.yml
conda activate graphnet
```

## Setup @Ext™

```bash
mamba env create -f utils/envs/graphnet_no_builds.yml
conda activate graphnet
```
